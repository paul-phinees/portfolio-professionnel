# Dossier Assets

Placez vos fichiers ici :

1. **photo.jpg** - Votre photo professionnelle (format JPG, PNG ou JPEG)
2. **cv.pdf** - Votre curriculum vitae au format PDF

## Instructions

1. Copiez votre photo professionnelle dans ce dossier et nommez-la `photo.jpg`
2. Copiez votre CV PDF dans ce dossier et nommez-le `cv.pdf`

Les fichiers seront automatiquement référencés dans le site web.
